Testing

Use the following methods to test

  • WAVE - Chrome Add On
  • AXE - within the inspector
  • Tab through the site
  • No ARIA roles within other ARIA roles
  • Screen reader
  • Skip to Main Content when tabbing
  • Able to tab within the menu.
  • BrowserStacks

Using Chrome Inspector

Select an element, like button or accordion item and then check 'Accessibility'. You should see the different aria states. If this is as expected then the aria is fine and working correctly.
